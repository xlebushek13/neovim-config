# My Neovim Config

## Instalation

- clone this repo to ~/.config/nvim
- after clone use PackerInstall dont worry about errors on first start neovim

## Architecture

- plugin options in after/plugin.
- keybindings in lua/maps.lua
- all plugins in lua/plugins.lua
